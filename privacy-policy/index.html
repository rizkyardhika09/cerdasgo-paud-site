Kebijakan Privasi CerdasGO PAUD dan TK

Berlaku untuk versi 1.0.0 · Diperbarui 21 Juli 2026

Ringkasnya: aplikasi menyimpan profil, progres, pengaturan, dan karya anak hanya di perangkat. Build saat ini tidak mengirim data pribadi atau aktivitas anak kepada developer maupun pihak ketiga.

Data yang disimpan di perangkat

CerdasGO dapat menyimpan nama panggilan anak, jalur belajar, teman belajar pilihan, progres aktivitas, ringkasan sesi, batas waktu bermain, status istirahat, karya gambar, thumbnail, dan pengaturan orang tua. PIN orang tua disimpan sebagai verifier aman, bukan sebagai teks PIN yang dapat dibaca.

Pengumpulan dan pembagian data

Build saat ini tidak mempunyai akun, server aplikasi, analytics, crash reporting pihak ketiga, cloud sync, tracking lintas aplikasi, atau transmisi data anak kepada developer. Data utama diproses secara lokal di perangkat.

Iklan, pembelian, dan kontribusi

Build saat ini tidak mempunyai iklan aktif, pembelian aktif, subscription, atau pembayaran aktif. Preview paket tidak dapat dibeli oleh anak. Alur kontribusi produksi tetap nonaktif dan tidak memberikan hadiah, badge, konten, atau keuntungan digital kepada anak.

Akses ke aplikasi eksternal

YouTube Kids hanya dapat dibuka setelah Parent Access dan konfirmasi eksplisit. Setelah aplikasi eksternal terbuka, kebijakan privasi dan kontrol aplikasi eksternal tersebut berlaku. CerdasGO tidak menginstal aplikasi lain secara otomatis.

Autentikasi perangkat

Face ID, fingerprint, atau autentikasi perangkat dapat digunakan untuk melindungi Area Orang Tua. Sistem operasi melakukan verifikasi. CerdasGO tidak menerima atau menyimpan template biometrik mentah.

Retensi dan penghapusan

Data tetap berada di perangkat sampai orang tua menghapus karya tertentu, menggunakan fitur Hapus Semua Data Lokal, menghapus aplikasi, atau sistem operasi menghapus penyimpanan aplikasi. Penghapusan seluruh data memerlukan autentikasi kuat dan konfirmasi kedua.

Perubahan kebijakan

Kebijakan ini harus diperbarui sebelum aplikasi menambahkan iklan, analytics, pembelian, akun, cloud sync, konten jarak jauh, support upload, atau pengiriman data keluar perangkat.
